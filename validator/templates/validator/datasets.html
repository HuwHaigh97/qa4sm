Datasets - Data to be validated

The following datasets can be validated within the QA4SM tool. These datasets are available in the "data" section of the validate page.

Dataset Details Versions supported Data availability Citation Filters (hover over for additional info)
C3S SM The C3S long-term data record provides active, passive and combined (active+passive) soil moisture data on a daily, dekadal and monthly basis. Observations are provided on a quarter degree grid (more information). The QA4SM tool supports version v201706 of the C3S product. Currently, the COMBINED product (combination of active and passive sensors) is supported. In future, it will be possible to download the C3S data from the climate data store. TBA
  • Valid range:
    Filter soil moisture values to ensure they are within the valid range (between 0 and 1 for the COMBINED product).
  • Use data where flag=0:
    Only use satellite observations where no inconsistencies were detected.
  • Frozen soil or snow-cover:
    Only use soil moisture data from grids cells without no snow cover or frozen soil (flag != 1).
  • Dense vegetation:
    Only use soil moisture data from grids cells without dense vegetation (flag != 2).
  • Ascending mode only:
    Only use satellite observations in ascending mode.
  • Descending mode only:
    Only use satellite observations from descending mode.
SMAP Level 3 SM The NASA SMAP level 3 data set provides global daily soil moisture observations at a 36 km resolution from the SMAP sensor (more information). The QA4SM tool supports version 5 of the SMAP L3 Soil moisture product. The data can be downloaded from the product homepage here. Click here
  • Valid range:
    Observations are filtered to ensure they are within the valid range (between 0 and 1)
H-SAF ASCAT SSM CDR The ASCAT dataset from the EUMETSAT H-SAF portal provides global daily soil moisture observations on a discrete global grid (DGG) with a spatial resolution of 25 km. Observations are provided from the ASCAT instrument on-board both Metop-A and Metop-B (more information). The QA4SM tool supports version H113 of the H-SAF ASCAT SSM CDR product. Datasets - Reference Data

The following datasets can be used as a reference within the QA4SM tool. These datasets are available in the "reference" section of the validate page.

Dataset Details Versions supported Data availability Citation Filters
ISMN The ISMN provides in-situ soil moisture data from over 2600 monitoring stations worldwide. The data is harmonised and quality controlled to ensure the highest quality data is provided through the portal. Data is provided from 1952 to present day (more information). The ISMN data is downloaded periodically for use in the QA4SM portal, therefore the version number (and data) will change over time. The current version used in v20180830. Data is freely available after registration here. Terms and conditions and readme (v20180830)
  • Quality flag is "good" (G):
    More information on the ISMN quality flags can be found here.
GLDAS GLDAS is a land surface model (LSM) which provides soil moisture data at different depths. Different versions are available for different time periods. For more information, visit the LDAS website: NASA Land Data Assimilation Systems Website Version 2.1 is supported within the QA4SM tool. This dataset covers the period 2000 to present day. Data is freely available after registration here for version 2.1 Click here
  • Frozen soil or snow-cover:
    Only use soil moisture data from grids cells where there is no snow cover or frozen soil.
